## Code

```java
class Solution {
    private int getValue(char c) {
        switch (c) {
            case 'I':
                return 1;
            case 'V':
                return 5;
            case 'X':
                return 10;
            case 'L':
                return 50;
            case 'C':
                return 100;
            case 'D':
                return 500;
            case 'M':
                return 1000;
            default:
                return 0;
        }
    }

    public int romanToInt(String s) {
        int n = s.length();
        int sum = 0;
        for (int i = 0; i < n; i++) {
            char curr = s.charAt(i);
            if ((i + 1 < n) && (getValue(curr) < getValue(s.charAt(i + 1)))) {
                sum -= getValue(curr);
            } else {
                sum += getValue(curr);
            }
        }
        return sum;
    }
}
// Time complexity: O(n)
// Space complexity: O(1)
```

## ðŸ”— [12. Integer to Roman](https://leetcode.com/problems/integer-to-roman/description/)

```java
class Solution {
    private static final int[] values = { 1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1 };
    private static final String[] symbols = { "M", "CM", "D", "CD", "C", "XC", "L", "XL", "X", "IX", "V", "IV",
            "I" };

    public String intToRoman(int num) {
        StringBuffer sb = new StringBuffer("");
        for (int i = 0; i < values.length; i++) {
            while (num >= values[i]) {
                sb.append(symbols[i]);
                num -= values[i];
            }
        }
        return sb.toString();
    }
}
// Time complexity: O(1)
// Space complexity: O(1)
```

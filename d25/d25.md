## ðŸ”— [540. Single Element in a Sorted Array](https://leetcode.com/problems/single-element-in-a-sorted-array/)

## Code

```java
class Solution {
    public int singleNonDuplicate(int[] nums) {
        int left = 0;
        int right = nums.length - 1;

        while (left < right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == nums[mid + 1]) {
                boolean isEven = (right - mid) % 2 == 1;
                if (isEven) {
                    right = mid - 1;
                } else {
                    left = mid + 2;
                }
            } else if (nums[mid] == nums[mid - 1]) {
                boolean isEven = (mid - left) % 2 == 1;
                if (isEven) {
                    left = mid + 1;
                } else {
                    right = mid - 2;
                }
            } else {
                return nums[mid];
            }
        }

        return nums[left];
    }
}
//Time Complexity: O(logn)
//Space Complexity: O(1)
```

class Solution {
    public int searchInsert(int[] nums, int target) {
        int s = 0;
        int e = nums.length - 1;
        while (s <= e) {
            int mid = s + (e - s) / 2;
            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] > target) {
                e = mid - 1;

            } else if (nums[mid] < target) {
                s = mid + 1;
            }
        }

        return s;
    }
}

## Code

###  Brute Force Approach : 

```java
class MedianFinder {
    List<Integer> list ;
    public MedianFinder() {
        list = new ArrayList<>();
    }
    
    public void addNum(int num) {
        list.add(num);
    }
    
    public double findMedian() {
        if(list.size()>0){
         Collections.sort(list);
         int middle = list.size()/2;
         if(list.size()%2==0){
            return (double) (list.get(middle-1) + list.get(middle))/2 ;
         }else{
            return (double) list.get(middle);
         }
        }
        return 0.0 ;
    } 
}

//Time Complexity: O(nlogn)
//Space Complexity: O(n)
```

### Optimised(Using PriorityQueue) : 

```java
class MedianFinder {

    Queue<Integer> maxHeap; // left pq
    Queue<Integer> minHeap; // right pq

    public MedianFinder() {
        maxHeap = new PriorityQueue<>(Comparator.reverseOrder());
        minHeap = new PriorityQueue<>();
    }
    
    //O(logn)
    public void addNum(int num) {
        if (maxHeap.isEmpty() || num < maxHeap.peek()) {
            maxHeap.offer(num);
        } else {
            minHeap.offer(num);
        }

        // If maxHeap has more than 1 element compare to minHeap
        if (maxHeap.size() > minHeap.size() + 1) {
            minHeap.offer(maxHeap.poll());
        }
        if (maxHeap.size() < minHeap.size()) {
            maxHeap.offer(minHeap.poll());
        }
    }
    
    //O(1)
    public double findMedian() {
        if (maxHeap.size() == minHeap.size()) {
            return (double) (maxHeap.peek() + minHeap.peek()) / 2;
        } else {
            return (double) maxHeap.peek();
        }
    }
}
```

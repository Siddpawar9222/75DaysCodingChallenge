## Using Recursion :

    private int recursion(int[][] grid, int i, int j) {
        int n = grid.length;
        int m = grid[0].length;
        if (i == n - 1 && j == m - 1) {
            return grid[i][j];
        }
        int minAns = Integer.MAX_VALUE;
        for (int dir[] : direction) {
            int i_ = i + dir[0];
            int j_ = j + dir[1];
            if (i_ >= 0 && i_ < n && j_ >= 0 && j_ < m) {
                minAns = Math.min(minAns, grid[i][j] + recursion(grid, i_, j_));
            }
        }
        return minAns;
    }
/*
Time Complexity: O(2^n)
Space Complexity: O(n)    // recursive stack
*/

## Using (Memoization) Top Down Approach :

```java
    private int memo(int[][] grid, int[][] memo, int i, int j) {
        int n = grid.length;
        int m = grid[0].length;
        if (i == n - 1 && j == m - 1) {
            return grid[i][j];
        }
        if (memo[i][j] != -1) {
            return memo[i][j];
        }
        int minAns = Integer.MAX_VALUE;
        for (int dir[] : direction) {
            int i_ = i + dir[0];
            int j_ = j + dir[1];
            if (i >= 0 && i_ < n && j_ >= 0 && j_ < m) {
                minAns = Math.min(minAns, grid[i][j] + memo(grid, memo, i_, j_));
            }
        }
        memo[i][j] = minAns;
        return memo[i][j];
    }
/*
Time Complexity: O(m*n)
Space Complexity: O(m*n)
*/
```

## Using (Tabulation) Bottom-Up Approach :

```java
        private int tabulation(int[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        int[][] dp = new int[n][m];

        //For n-1 i can move right only and for m-1 i can move bottom only
        //For n-1 row(right to left) and m-1(bottom to up)
        int sum = 0;
        for (int j = m - 1; j >= 0; j--) {
            sum += grid[n - 1][j];
            dp[n - 1][j] = sum;
        }
        sum = 0;
        for (int i = n - 1; i >= 0; i--) {
            sum += grid[i][m - 1];
            dp[i][m - 1] = sum;
        }

        for (int i = n - 2; i >= 0; i--) {
            for (int j = m - 2; j >= 0; j--) {
                dp[i][j] = grid[i][j] + Math.min(dp[i][j + 1], dp[i + 1][j]);
            }
        }
        //printArray(dp);
        return dp[0][0];
    }
/*
Time Complexity: O(m*n)
Space Complexity: O(m*n)
*/
```

## Using Recursion :

```java
    private boolean recursion(String s ,Set<String> set, int idx){
        int n = s.length();

        if(idx==n){
             return true ;
        }

        for(int i = idx ;i<n ;i++){
            String subString = s.substring(idx,i+1);
            if(set.contains(subString)&& recursion(s,set,i+1)){
                 return true ;
            }
        }
        return false ;
    }

    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> set = new HashSet<>(wordDict);
        return recursion(s,set,0);
    }
/*
Time Complexity: O(2^n)
Space Complexity: O(n)    // recursive stack
*/
```

## Using (Memoization) Top Down Approach :

```java
class Solution {

    static boolean isPalindrome(String str, int i, int j) {
        while (i < j) {
            if (str.charAt(i) != str.charAt(j))
                return false;
            i++;
            j--;
        }
        return true;
    }

    static int minPalPartition(String str, int i, int j, int[][] memo) {
        if (i >= j) {
            return 0;
        }
        if (isPalindrome(str, i, j)) {
            return 0;
        }

        if (memo[i][j] != -1) {
            return memo[i][j];
        }

        int minCuts = Integer.MAX_VALUE;

        for (int k = i; k < j; k++) {
            int left, right;

            if (memo[i][k] != -1) {
                left = memo[i][k];
            } else {
                left = minPalPartition(str, i, k, memo);
                memo[i][k] = left;
            }

            if (memo[k + 1][j] != -1) {
                right = memo[k + 1][j];
            } else {
                right = minPalPartition(str, k + 1, j, memo);
                memo[k + 1][j] = right;
            }

            int cuts = left + right + 1;
            minCuts = Math.min(minCuts, cuts);
        }

        memo[i][j] = minCuts;
        return memo[i][j];
    }

    static int palindromicPartition(String str) {
        int n = str.length();
        int[][] memo = new int[n][n];
        for (int[] row : memo) {
            Arrays.fill(row, -1);
        }
        return minPalPartition(str, 0, n - 1, memo);
    }
}
/*
Time Complexity: O(n^3)
Space Complexity: O(n^2)
*/
```

## Using (Tabulation) Bottom-Up Approach :

```java
    private boolean tabulation(String s ,Set<String> set){
        int n = s.length();
        Boolean [] tab = new Boolean[n+1];
        tab[n]=true ;

        for(int idx=n-1 ;idx>=0 ;idx--){
            tab[idx]=false;
            for(int i = idx ;i<n ;i++){
                String subString = s.substring(idx,i+1);
                if(set.contains(subString)&& tab[i+1]){
                     tab[idx] =true ;
                     break ;
                }
            }
        }
        //System.out.println(Arrays.toString(tab));
        return tab[0] ;
    }
/*
Time Complexity: O(n*n)
Space Complexity: O(n)
*/
```
